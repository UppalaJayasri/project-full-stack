# Indian Heritage Auth App 🏛️

A comprehensive web application for exploring Indian heritage sites with role-based authentication, interactive features, and educational content.

## 🌟 Features

### **User Roles:**
- **Regular User (Explorer)** - Browse heritage sites, take quiz, watch videos, submit feedback
- **Cultural Enthusiast** - All user features + contribute articles, earn badges
- **Admin** - Full dashboard for user, content, and feedback management

### **Key Features:**
- 🗺️ **Interactive Map** - Explore 15 heritage sites across India
- 📹 **Travel Videos** - YouTube guides for Taj Mahal, Red Fort, Golden Temple, etc.
- 🎓 **Educational Quiz** - 10 questions about Indian heritage
- 💬 **Feedback & Comments** - Rate app and discuss with community
- 🏆 **Achievement System** - Badges for cultural enthusiasts
- 📊 **Admin Dashboard** - Manage users, content, and interactions

## 🎯 Usage

1. **Welcome Page** - Click "Continue"
2. **Select Role** - Choose Admin, Cultural Enthusiast, or Explorer
3. **Login/Signup** - Enter credentials (demo mode - any password works)
4. **Explore** - Navigate through tabs:
   - Overview - App introduction
   - Map - Interactive India map
   - Gallery - 15 heritage sites with details
   - Quiz - Test your knowledge
   - Feedback - Rate and comment (Users only)
   - Contribute - Submit content (Cultural Enthusiasts only)
