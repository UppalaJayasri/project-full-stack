# Indian Heritage Explorer - Complete Features

## User Roles & Access System

### 1. **Regular User**
- Login with any email (e.g., `user@heritage.com`)
- Access to all heritage sites information
- Interactive map exploration
- Educational quiz (10 questions)
- View gallery of 15 heritage sites
- Read detailed information about each site

### 2. **Cultural Enthusiast** 👑
- Login with email containing "enthusiast" or "expert" (e.g., `enthusiast@heritage.com`)
- All regular user features PLUS:
- **Contribute Content**: Write articles, reviews, and guides
- **Track Contributions**: Monitor published content and likes
- **Earn Achievements**: Unlock badges for community contributions
- **Special Badge**: Purple "Cultural Enthusiast" badge displayed
- **Contribution Portal**: Dedicated section for content creation

### 3. **Admin** 🛡️
- Login with email containing "admin" (e.g., `admin@heritage.com`)
- **Complete Dashboard** with:
  - User Management (view, edit roles, delete users)
  - Content Management (publish/unpublish, edit, delete)
  - Interaction Moderation (approve/reject comments, reviews, questions)
  - Real-time Statistics (users, enthusiasts, content, quiz scores)
  - Role Assignment (promote users to Cultural Enthusiast or Admin)
